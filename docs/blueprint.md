# **App Name**: SpendWise

## Core Features:

- Transaction Input: Allow users to input income and expenses, categorizing each transaction (e.g., groceries, rent, entertainment).
- Visualization: Display spending trends and patterns using charts (e.g., bar, pie).
- Balance Display: Display the sum of income, the sum of expenses, and the remaining budget.
- Category Suggestions: Suggest a budget category based on a transaction description provided by the user. Use an LLM as a tool to help make the decision.
- Goal Tracking: Allow users to create and track financial goals, such as saving for a vacation.

## Style Guidelines:

- Primary color: Vivid blue (#4285F4) to convey trust and stability.
- Background color: Light blue (#E3F2FD) for a clean and calming feel.
- Accent color: A vibrant green (#34A853) to highlight positive actions like adding income or achieving a goal.
- Body and headline font: 'PT Sans', a humanist sans-serif providing a modern and accessible feel, is suitable for both headlines and body text.
- Use simple, outlined icons for categories and menu options.
- A clean, card-based layout for transactions and summaries.
